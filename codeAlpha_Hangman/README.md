# 🎮 Hangman Game

A simple console-based Hangman Game developed using Python.

## Features

- Random word selection
- ASCII Hangman graphics
- Input validation
- Tracks guessed letters
- Maximum 6 incorrect guesses
- Play Again option
- Beginner-friendly code structure

## Technologies

- Python 3

## Project Structure

```
main.py
game.py
utils.py
words.py
```

## How to Run

Clone the repository.

```
git clone https://github.com/yourusername/CodeAlpha-Hangman.git
```

Go inside the folder.

```
cd CodeAlpha-Hangman
```

Run

```
python main.py
```
